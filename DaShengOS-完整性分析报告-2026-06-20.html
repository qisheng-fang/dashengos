5 多个 0B 数据库文件 data/dasheng [2-5].db 版本迁移残留 清理无效 DB 文件
6 测试覆盖率不足 全局 后端 API 无单测，前端仅 e2e 添加 Vitest 单测 + API 集成测试
7 pnpm-lock 多版本 根目录 3 个 lock 文件 依赖版本不确定 统一为 pnpm-lock.yaml

🟢 低风险项

  • DeerFlow yaml 配置硬编码路径，部署需手动调整
  • Sandbox 仅支持 Linux (seccomp/namespace)，macOS 开发受限
  • 旧 Worker (sau-bridge/douyin-bridge 等) 未纳入 v0.3 管理
  • backups/ 目录不可见，备份策略不明确
